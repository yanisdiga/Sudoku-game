# Sudoku Game
An Sudoku Game developped in C with ncurses

# Installation
Notice : you must have ncurse lib
```
gcc sudoku.c -o sudoku -lncurses
```
# Bind
- "space" -> add a number
- "x" or "0" -> clear the box
- "h" -> solve the case
- "s" -> solve the game
- "r" -> reset grid
